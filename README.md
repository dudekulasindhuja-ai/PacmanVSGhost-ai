# 🎮 Pacman AI Game using Minimax & Alpha-Beta Pruning

## 📌 Description
This project implements an intelligent game-playing agent using the **Minimax algorithm with Alpha-Beta pruning**.  

The game is a grid-based **Pacman vs Ghost** environment:
- Pacman is controlled by the user
- Ghost is controlled by AI

The AI predicts future moves and selects the optimal action.

---

## 🧠 Core Concepts Used
- Game Trees
- Minimax Algorithm
- Alpha-Beta Pruning
- Evaluation Function

---

## 🎯 Features
- Interactive grid-based game using **Pygame**
- AI-controlled ghost
- Multiple difficulty levels (Easy, Medium, Hard)
- Score system with dot collection
- Maze with walls
- Start menu, restart, and home options
